# 🔍 Algorithm Problem Solving (Baekjoon)

## 📌 개요
* **언어:** C++
* **난이도:** Bronze 5 ~ Silver 3
* **목표:** 알고리즘적 사고방식 함양 및 C++ 문법 숙달

## 💡 학습 포인트 & 최적화 팁

### 🚀 입출력 성능 개선 (Fast I/O)
* **실행 시간 단축:** `endl` 대신 `'\n'` 사용. (`endl`은 개행과 동시에 버퍼를 비우는 과정이 포함되어 매우 느림)
* **입출력 가속:** `ios::sync_with_stdio(0);`와 `cin.tie(0);` 사용.
    * `sync_with_stdio(0)`: C와 C++ 입출력 시스템 동기화를 해제하여 성능 향상.
    * `cin.tie(0)`: `cin`과 `cout`의 연결을 끊어 입력 시 자동 flush 방지.
    * **주의사항:** 가속 사용 시 `printf`, `scanf` 등 C 표준 입출력 함수 혼용 금지.

### 🧩 논리 설계 및 전략
* **시간 계산:** 시, 분, 초 계산 문제는 최소 단위(초)로 통일하여 계산한 뒤 다시 변환하는 것이 가장 정확함.
* **데이터 변환:** 아스키코드를 활용하여 문자(`char`)와 정수(`int`) 간의 유연한 입출력 변환.
* **실무 개발 vs 코딩 테스트:**
    * 코딩 테스트는 깔끔한 코드보다 **빠르고 정확한 구현**이 우선.
    * 복잡한 디버거 사용보다는 논리적인 흐름 추적(Print Debugging 등) 중심.
    * 출력의 마지막 공백이나 줄바꿈은 정답 판정에 영향을 주지 않으므로 구현 효율에 집중.
